Elemen yang diterapkan

Pada website ini, saya menerapkan beberapa interaksi sederhana menggunakan JavaScript untuk memberikan pengalaman yang lebih dinamis bagi pengguna:

  • Responsive Navigation: Pada layar kecil, tombol menu dapat ditekan untuk menampilkan atau menyembunyikan navigasi.
  • Image Slider: Bagian Gallery memiliki slider gambar yang dapat dikendalikan dengan tombol prev/next serta berjalan otomatis setiap beberapa detik.
  • Ancient Mode: Tombol â€œAncient Modeâ€ di bagian header memungkinkan pengguna mengubah tampilan website menjadi tema Mesir Kuno.

Seluruh interaksi di atas dibuat menggunakan JavaScript murni tanpa library tambahan.
